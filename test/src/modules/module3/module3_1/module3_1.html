IE8兼容
IE8兼容仅采用了babel进行兼容,并且做了按需加载兼容，避免全局污染
按需兼容通过以下插件来实现
  • babel-plugin-transform-runtime
  • babel-runtime
  • @babel-polyfill
  • babel-preset-env
原理：babel会自动检测代码中使用的新特性，通过指定需要兼容的浏览器版本，给予相关的polyfill
参考链接1 参考链接2
例如：创建一个test.js文件 在内部写入

            var map = new Map();
编译出来的文件:
只引入了map的polyfill以及一个公共的polyfill (require(192))
现在添加一个Promise

            var promise = new Promise();
            var map = new Map();
编译出来的文件:
只多引入了一个Promise的polyfill
